# Q0. Add the numbers
You are given **N** integers. Please find the sum of them.

## Input
The first line contains a number **N** `(1 ≤ N ≤ 1000)`, the number of integers. <br>
The second line contains **N** integers. <br>

Example:
```
5
1 2 3 4 5
```

## Output
The sum of the integers in line 2.

Example:
```
15
```

Explanation: <br>
1 + 2 + 3 + 4 + 5 = 15
